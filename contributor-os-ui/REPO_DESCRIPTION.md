## Long Description

```
Contributor OS is a real-time operating system designed specifically for open-source contributors. It provides a unified dashboard to track GitHub activity, manage tasks linked to PRs/issues, visualize contributions on a calendar, take notes, and receive real-time notifications. Built on an event-driven architecture where everything revolves around events - GitHub webhooks, user actions, and contributions all flow through a unified event stream. Features include GitHub OAuth authentication, public/private dashboards, open-source repository filtering, timeline views, task management, and more. Built with Next.js 16, TypeScript, Tailwind CSS, Zustand, and TanStack Query.
```
